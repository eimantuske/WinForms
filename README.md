# WinForms Kolegijos Darbai / College Projects

Ši saugykla yra skirta mano studijų metu atliekamiems C# Windows Forms projektams. Čia kaupiami visi laboratoriniai darbai, užduotys ir kiti programavimo pavyzdžiai.

---

## Lietuviškai

### Apie saugyklą
Visi projektai yra sugrupuoti į atskirus aplankus pagal jų paskirtį. Tai mano mokymosi archyvas, kuriame matoma pažanga studijuojant C# programavimą ir grafinės sąsajos kūrimą.

### Projektų sąrašas
* **Math Quiz** – loginis matematikos žaidimas su laiko skaičiavimu.
* **Pabandymas** – pirmieji žingsniai ir bazinių komponentų testavimas.

### Įrankiai
* **Kalba:** C#
* **Aplinka:** .NET / WinForms
* **IDE:** Visual Studio 2022

### Kaip naudotis?
1. Atsisiųskite arba klonuokite šią saugyklą.
2. Eikite į pasirinkto projekto aplanką.
3. Atidarykite .slnx arba .sln failą naudodami Visual Studio.
4. Paleiskite projektą (Start).

## English

### About this Repository
This repository is dedicated to my C# Windows Forms college assignments. It serves as a collection of laboratory works, tasks, and various programming examples created during my studies.

### Project List
* **Math Quiz** – A mathematical logic game with a countdown timer.
* **Pabandymas** – Initial tests and experimentation with basic UI components.

### Technologies
* **Language:** C#
* **Framework:** .NET / WinForms
* **IDE:** Visual Studio 2022

### How to Run
1. Download or clone this repository.
2. Navigate to the specific project folder.
3. Open the .slnx or .sln file using Visual Studio.
4. Run the project (Start).
